其实在最开始的时候，就已经提示`An error occurred while installing yarl==1.3.0`, 这个时候，如果我们看到了，应该要停止安装接下来的包，因为，最后还是会报错的。

一个简单有效的思路，就是手工安装一下 yarl 包，看是不是版本不对。
